# XGBoost-in-Healthcare
The application of XGBoost model in healthcare industry.

## Overview
This is a preliminary study on the application of XGBoost using Jupyter Notebook and scikit-learn library.
The dataset named Wisconsin Breast Cancer, is the public dataset obtained from UCI Machine Learning
Repository.
https://archive.ics.uci.edu/ml/datasets/breast+cancer+wisconsin+(diagnostic)
